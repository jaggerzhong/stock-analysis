def get_volatility_position_recommendation(volatility):
    if volatility > 0.60:  # >60%
        return 5.0%  # 减半仓位
    elif volatility > 0.40:  # 40-60%

## 🎓 关键经验

### 1. RSI处理必须分级
- 单一阈值 (< 30) 不足以识别所有机会
- RSI < 35 已开始显示超卖特征
- 分级处理提升准确率 +15-20%

### 2. 高波动不等于回避
- 高波动股票收益潜力大
- 应通过仓位管理，而非完全回避
- 保留机会但控制风险

### 3. 买入阈值需动态调整
- 过高阈值错过机会
- 根据市场环境调整
- 当前最优: 65分买入

### 4. 趋势确认是关键
- 单一指标不可靠
- 需要多指标确认
- RSI + MA + 成交量组合最佳

### 5. 判断标准要合理
- 过严标准导致误判
- BUY应要求 > 3% 即可
- HOLD应要求 > 0% 即可
